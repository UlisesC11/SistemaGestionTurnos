# Anexo - Principios SOLID

El propósito del principio SOLID es mejorar la calidad del diseño del software orientado a objetos, haciendo que el código sea más mantenible, reducir la complejidad de añadir nuevas funcionalidades, aumentar la reusabilidad de piezas y componentes, mejorar la calidad del código y su compresión. Cada letra de SOLID representa un principio que ayuda a evitar problemas comunes como el acoplamiento fuerte, la rigidez o la fragilidad del software cuando se hacen cambios.
